What is Ruria?

Ruria is a cute discord bot with a lot of fun and useful commands. It also has moderation commands for your server and a global leveling system!

What Ruria has or can do?

Ruria has a lot of Fun, Moderation, Utility and Roleplay commands!

1

Ruria Leveling System

  • - Gain random exp for message sent every minute
  • - Custom rank cards coming soon
  • - Custom levelup cards coming soon
  • - Get currency for each level up
  • - A leaderboard for highest level users

2

Ruria Economy System

  • - Earn daily and compete for the leaderboard
  • - Jobs and gambling commands for earning ruia (currency)
  • - Compete with all other players globally!
  • - Custom badges soon to be added!
  • - 5 professions to play with each with its own difficulty!

3

Useful Commands

  • - Animequote (With Image)
  • - Custom Welcomer coming soon!
  • - Customizable setembed command with/without images!
  • - Tons of useful commands like "randomcolor", "poll", "userinfo", "serverinfo", "purge", "rps", "slots", "calculator", and a lot more!
